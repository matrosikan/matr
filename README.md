1. Задание:
Напишите рекурсивную функцию для вычисления суммы цифр числа.

2. Алгоритм:
Рекурсивная функция:
Базовый случай: если число < 10, возвращаем само число
Рекурсивный случай: последняя цифра + сумма цифр оставшейся части
Функция последовательно отделяет последнюю цифру(n % 10) и рекурсивно обрабатывает оставшееся число, пока не останется одна цифра: Функция sum_digits вызывает саму себя внутри своего тела, при каждом вызове число уменьшается: n // 10, к последней цифре n % 10 прибавляется результат вызова этой же функции для уменьшенного числа. Затем суммирует все полученные цифры при возврате из рекурсии.

3. Временная сложность: O(log n)

4. Объснение временной сложности:
1) Алгоритм обрабатывает каждую цифру числа ровно один раз
2) Количество цифр в числе n равно log₁₀n + 1
3) Каждая цифра обрабатывается за константное время O(1)
4) Общее время работы пропорционально количеству цифр: O(log n)

6.
DFS (в глубину):
1) Идёт по одному пути до конца, потом возвращается
2) Использует стек
3) Меньше памяти для глубоких деревьев

BFS (в ширину):
1) Идёт уровень за уровнем
2) Использует очередь
3) Находит кратчайший путь
4) Больше памяти для широких деревьев
